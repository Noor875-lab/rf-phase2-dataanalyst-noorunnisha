## Project Summary
This project demonstrates an end-to-end data analysis workflow including data cleaning, SQL querying, dashboard visualization, and business insight generation. The goal is to transform raw business data into meaningful insights for decision making.

## Work Done
- Created GitHub repository and organized folder structure  
- Set up Python and Jupyter in VS Code  
- Loaded dataset using Pandas  
- Checked missing values and duplicates  
- Cleaned and prepared data for analysis  
- Performed exploratory data analysis  
- Answered business questions using Python

## Business Questions & Insights
1. Identified top revenue-generating stores.  
2. Found best-selling products based on quantity.  
3. Analyzed revenue contribution by aisle/category.  
4. Studied monthly sales trends using transaction dates.  
5. Evaluated the impact of discounts on final sales.
